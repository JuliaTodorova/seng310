UVic Course Registration System Redesign

Project Mandate

The UVic Course Registration System is inadequate in its current form and needs to be redesigned to tailor the student’s needs. The new model should be user friendly, which allows students to search for courses fast and efficiently and provides them with all the information they need to make educated decisions regarding their studies. It should be designed in such that less technical-savvy users and advanced users will both use it seamlessly. The page should be easily accessible with the important information front and center. The website should not have multiple pages for user input or selection when searching for courses, as everything will be interactive and the user will not have to leave or reload the page for the data to be presented.

User Requirements

The system should be
  • User friendly
  • Fast and efficient
  • Simplistic
The system would be able to
  • Search courses
  • Select courses from a drop down menu
  • Show a weekly calendar
  • Simplify registration
System features
  • Drag and drop courses into calendar
  • Allow users to register from the calendar menu
  • Only show courses where the prerequisites are met
  • View waitlisted courses on top of registered course calendar
  • Review and rate courses
  • Rate professors
  • Recommend courses based on GPA, previously taken courses and highly rated courses

Review and rate courses

Andrea Thomas just finished PSYC 412A and immensely enjoyed the course. She would like to write a review to share her experiences and thoughts with other students. She logs in to the UVic website and navigates to the course page, where all the courses she has previously taken are listed. She selects PSYC 412A, where she then can write a course review. After Andrea has written the review, she is prompted to rate the course out of five and state weather she would recommended it to other students. Andrea rated the course five stars and would recommended the course to her peers.

Rate professors

Martin Larson needed a 2nd year elective and decided to take HLTH 251. He heard many great things about Dr. Charlotte Loppie, the professor that would be teaching the course. After he took the course, he thought that the Dr. Loppie was a phenomenal professor, who used amusing stories to entice the class. View waitlisted courses on top of registered course calendar

Martin Larson registered for six courses and was put on the waitlist for an additional three. He plans to drop between three and four courses but wants to see what the schedule for his courses will look like accounting the waitlisted courses. Martin goes on to the UVic websites and logs into his account, where he navigates to the course registration page. He then, checked the button called “Show waitlisted courses” under the Search options on the right-hand side of the page. All his waitlisted courses will show on top of his registered courses in the weekly calendar. The waitlisted courses will supply the same information as the registered courses, but will have a transparent background to be able to differentiate between them. Martin can now see all his registered courses and waitlisted courses in one spot, allowing him to make alternation to his schedule easily.

He looks through the list of courses that have appeared until he sees the one that he is looking for. He then drags and drops Economics 111 into the calendar. It automatically snaps into the T/W/F 12:30-1:20 timeslot.

He clicks on the arrow on the left side and finds it returns the calendar to the previous week.

Hierarchical Task Analysis

Review and rate courses

  1. Assume login access into the UVic website
  2. Navigate to the course page
  3. Select a course for review
  4. Write a review for the course taken
  5. Rate the course out of five
  6. State weather it should be recommended to other students

Rate professors

  1. Assume login access into the UVic website
  2. Navigate to the course page
  3. Select a course
  4. Navigate to the professor information
  5. There is an option to either write a review or simply rate the professor out of five
  6. Write review or rate professor

View waitlisted courses on top of registered course calendar

  1. Assume login access into UVic website
  2. Navigate to the Course Registration page
  3. Find the View Waitlist button
  4. Click on the Waitlist button
  5. Look to the left side of the page where the weekly calendar is displayed
  6. On top of the registered courses will be a transparent view of each of the courses in which the user is waitlisted
  7. The user can now compare the waitlisted courses and times

Drag and drop into calendar

  1. Assume login access into the UVic website
  2. Navigate to the Course Registration page
  3. The user can search for courses on the right side of the screen under "course search"
  4. The user will select the term they want
  5. The user will type in the course they are looking for by subject
  6. The user will click on the course that they want and drag it over to the calendar

Show weekly calendar

  1. Assume login access into the UVic website
  2. Navigate to the Course Registration page
  3. The calendar appears on the left side of the screen
  4. The user can select the term they want on the right side of the screen
  5. The user can click arrows that appear on the left and right side of the calendar to cycle through the weeks

Make registration simple

  1. Assume login access into the UVic website
  2. Navigate to the Course Registration page
  3. The user can type the course into the box at the bottom of the page and press "register"
  4. The user can also register from the course search box. They search for the course that they want and there is a button on the course that allows them to register

Recommend courses based on GPA, previously taken courses and highly rated courses

  1. Assume login access into the UVic website
  2. Navigate to the Course Registration page
  3. The recommended courses will appear in a box underneath the course search box
  4. The user can click on the course and drag it into the calendar and it will snap into the right timeslot

Searching for courses

  1. Assume login access into UVic website
  2. Navigate to the Course Registration page
  3. Select the semester in right window
  4. Type the name of the course in the search bar on the right side of the screen
  5. Select course from the drop down menu

Register in calendar window

  1. Assume login access into UVic website
  2. Navigate to the Course Registration page
  3. Choose a course to register in
  4. Click on the ‘register’ button next to the course in the drop down menu

Only show courses which pre-requisites are met

  1. Assume login access into UVic website
  2. Navigate to the Course Registration page
  3. Check the ‘Show courses available to me’ box on the right side of screen
  4. Results will only include courses in which the user has met the pre-requisites to

Drop down course search menu

  1. Assume login access into UVic website
  2. Navigate to the Course Registration page
  3. Type letters into search bar on the right side of the screen
  4. Letter by letter the drop down bar refreshes its search results
